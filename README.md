# GL Chat Platform

Great Lakes Chat Widget — lead capture widget for senior living communities.

## Features
- Conversational chat widget (Web Component with Shadow DOM)
- - Pricing tables, tour scheduling, floor plan galleries
  - - Admin dashboard for managing communities and leads
    - - Vercel serverless API backend
     
      - ## Deploy
      - Import this repo on Vercel — it auto-deploys with `vercel.json` config.
